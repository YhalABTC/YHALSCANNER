## 업데이트 방법
1. GitHub 리포(YHALSCANNER) 접속
2. **Add file → Upload files**
3. 이 폴더의 **모든 파일**을 드래그 (같은 이름은 자동 덮어쓰기)
4. 아래 **Commit changes** 클릭
5. 1~2분 뒤 반영 — 폰에서는 **강력 새로고침** 필요
